## Mobile Mini-Game

You can consult and practice the two syllabaries used in Japanese writing: Hiragana and Katakana.

The styles are inspired by [this video clip](https://www.youtube.com/watch?v=-GlK_8qAM8Q&ab_channel=Mitski).
